# Code of Conduct

This project adopts the [Contributor Covenant
v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful and constructive.
- Argue with evidence, not people.
- Assume good faith. Ask questions before correcting.
- No harassment, no discriminatory language, no personal attacks.

## Reporting

If you experience or witness behaviour that violates this code, please file a
private report by emailing `conduct@opendomaindata.org`. Reports are read by
the maintainers and handled confidentially.
